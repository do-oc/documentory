# Doctor

## Install

### Mac 

```
brew install python
pip install pdm
pdm install
```
### Windows

Install Python https://www.python.org/downloads/windows/

```
pip install pdm
pdm install
```

## Usage

```
pdm run python -m bin.cli
```

Then input the path to your project to analyze. 
For large project it can take couple of minutes
